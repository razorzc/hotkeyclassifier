# PowerLineCV — 图像数据集构建工具

面向 CV 数据集构建的桌面应用，用于电力巡检、工业缺陷图像的快速筛选、分类、裁剪与标注。

---

## 完整操作

### 浏览导航

| 按键 | 功能 |
|------|------|
| ← / → | 上一张 / 下一张 |
| Home / End | 第一张 / 最后一张 |
| Ctrl+滚轮 / +/- | 缩放 |
| 0 | 适应窗口 |
| Ctrl+1 | 原始尺寸 |
| 中键拖动 / Space+左键 | 平移 |

### 图片分类

| 按键 | 标签 | 输出目录 |
|------|------|---------|
| **Q** | Line_Broken_strands | 输电线/断股 |
| **W** | Line_Damage | 输电线/损伤 |
| **E** | Line_Foreign_object | 输电线/异物 |
| **R** | Line_Normal | 输电线/正常 |
| **A** | Tower_Deformation | 塔杆/变形 |
| **S** | Tower_Corrosion | 塔杆/锈蚀 |
| **D** | Tower_Foreign_object | 塔杆/异物 |
| **F** | Tower_Normal | 塔杆/正常 |
| **T** | 待定 | uncertain |

分类时只写入元数据（增量 `.json` 文件），不移动原图。

### 缺陷裁剪

| 按键 | 功能 |
|------|------|
| **1** | 裁剪框 224×224 |
| **2** | 裁剪框 384×384 |
| **3** | 裁剪框 640×640 |
| **Enter** | 独立保存裁剪（不分类） |
| 分类键+裁剪框 | 分类同时保存裁剪 |

### 批次管理

分类可按批次组织，便于分批导出和管理。

| 操作 | 位置 | 功能 |
|------|------|------|
| 查看当前批次 | 工具栏 `批次: xxx ▾` | 显示当前活动批次 |
| 切换批次 | 下拉 → 点击批次名 | 后续分类归入所选批次 |
| 新建批次 | 下拉 → `+ 新建批次...` | 创建新批次并切换 |
| 管理批次 | 下拉 → `管理批次...` | 批次列表、删除（含清理元数据） |
| 按批导出 | Ctrl+E → 勾选批次 | 只导出选中批次的待导出记录 |

- 首次启动自动创建「默认」批次
- 隔天启动恢复上次活跃批次，不自动切换
- 删除批次同时清理该批所有元数据

### 导出

| 操作 | 功能 |
|------|------|
| **Ctrl+E** | 弹窗选择批次，将待导出图片批量复制到目标目录并重命名为 `{label}_{6位流水号}.jpg` |

### 撤销与筛选

| 操作 | 功能 |
|------|------|
| **Ctrl+Z** | 撤销最近一次分类 |
| 工具栏 `≡ 撤销列表` | 查看并撤销本人的任意分类记录 |
| 工具栏 `∨ 仅未分类` | 只显示尚未分类的图片 |

### 进度恢复

- 打开文件夹自动读取 `.sort_progress.{用户名}.json`，恢复到上次浏览位置
- 同时合并所有用户的进度文件，显示已分类状态

## 分类信息显示

浏览图片时，左上角浮层显示：

| 状态 | 显示 |
|------|------|
| 未分类 | 文件名、分辨率、缩放、序号 |
| 已分类（1人） | `Line_Broken_strands \| zjc \| 2025-06-25` |
| 已分类（多人） | `[3人标注] Line_Broken_strands \| zjc \| 2025-06-25` |

---

## 多人协作

1. 首次启动弹窗设置用户名
2. 每人的分类记录写入独立增量文件 `labels/.entries/{用户}_{时间}.json`，互不冲突
3. 进度文件按用户拆分 `.sort_progress.{用户}.json`，合并读取
4. 同一张图被多人分类后，信息浮层显示标注人数

## 项目结构

```
sort/
├── main.py              # 入口
├── config/              # 配置文件
│   ├── config.json      # 快捷键、分类、路径
│   └── dark_theme.qss   # 深色主题
├── core/                # 核心逻辑
│   ├── image_manager.py   # 图片索引、加载、筛选
│   ├── entry_manager.py   # 增量元数据读写
│   ├── batch_manager.py   # 批次索引管理
│   ├── label_manager.py   # CSV 汇总（兼容旧版）
│   ├── undo_manager.py    # 撤销栈
│   ├── crop_manager.py    # 裁剪保存
│   ├── preload_manager.py # 异步预加载
│   ├── progress_manager.py# 进度持久化（多用户）
│   ├── settings.py        # 配置读写
│   ├── updater.py         # 版本检查
│   └── blur_detector.py   # 模糊检测（可选）
├── widgets/             # UI 组件
│   ├── image_viewer.py    # QGraphicsView 查看器
│   ├── crop_overlay.py    # 裁剪框
│   ├── drag_zone.py       # 拖拽分类区
│   ├── info_overlay.py    # 信息浮层
│   ├── directory_panel.py # 目录树（已弃用）
│   └── thumbnail_strip.py # 缩略图列（已弃用）
├── ui/                  # 界面
│   ├── main_window.py     # 主窗口
│   └── settings_dialog.py # 设置（快捷键/分类/裁剪可编辑）
└── utils/
    └── logger.py
```

## 技术栈

Python 3.12 / PySide6 / Pillow / numpy

OpenCV 为可选依赖（用于模糊检测，exe 中不打包）。
